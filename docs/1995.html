编程

从整数列表中，计算每个元素的出现次数，并添加项目，作为子列表计数。

输入列表:inp = [7，9，7，4，3，5，3，6，9，3]
输出列表:out= [[7，2]，[9，2]，[4，1]，[3，3]，[5，1]，[6，1]]

#注意:其中7是列表项，2是出现次数

使用计数器()方法

from collections import Counter
def count_items(inp):
     coll = Counter(inp)
     out = []
     for key,val in coll.items():
          out.append([key,val])
 return out
