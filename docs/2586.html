转换日期

用于加密数据的API以串行格式返回日期，该格式由一个整数组成，用于测量自1970年1月1日以来经过的毫秒数。

该日期格式可能看起来像例如1644532457000。这实际上是针对2022年2月10日星期四23:34:19 GMT+0100(中欧标准时间)。

使用下面的函数，您可以在JavaScript中将以毫秒为单位的日期转换为人类可读的时间:

function serialDateToNiceDate(dateinserialformat){
 return new Date(dateinserialformat)
}

您可能还需要相反的东西，即将给定的日期从nice格式转换为serial格式。这很简单，因为它是由一个本地JavaScript对象处理的:

Date.parse(dateinhumanreadableformat)

 }var sum1sq = 0, sum2sq = 0

 }var tmp = 0
