## 5.使用伪标记化

在自然语言处理(NLP)领域中完成的大多数文本分析都涉及标记化的过程。标记化将文本分解成标记，也称为单词或短语。伪标记化是用于将文本分解成更小部分的过程，以便它可以用于自然语言处理(NLP)方法。伪标记化背后的基本思想是用伪标记替换一小部分文本。伪标记是用于替换较小部分原始文本的唯一标记。伪标记化用于文档分类、聚类和类似的任务。
