# 介绍

NumPy 代表数字 Python。NumPy 是 Python 分发平台 Anaconda 提供的一个包。NumPy 库是一个丰富的模块，可以帮助我们处理多维数组对象。它允许用户构建更高维的矩阵来执行复杂的数学计算。NumPy 内置了许多数学运算、线性代数和傅立叶变换相关的函数。

# NumPy 的目的

*   基本数学运算、统计计算和基本线性代数。
*   构造一个 n 维数组和矩阵
*   访问、更新数组中的元素。
*   重新排列或重组矩阵的索引。
*   分割矩阵。

# NumPy 的优势

*   内存消耗更少
*   快速计算能力
