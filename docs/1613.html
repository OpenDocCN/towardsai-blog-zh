安装和设置

安装顶部刮泥板时，使用

pip install top-github-scraper

添加凭据

请注意，在没有凭证的情况下使用top-github-scraper完全没问题。但是你每小时最多只能提出60个请求。通过添加您的凭证(用户名和令牌)，您可以每小时发出多达5000个请求。

为了确保你能收集到许多存储库和用户，把你的GitHub的证书添加到根目录下的.env文件中。

touch .env

将您的用户名和令牌添加到.env文件中:

GITHUB_USERNAME=yourusername
GITHUB_TOKEN=yourtoken
