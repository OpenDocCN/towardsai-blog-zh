Google Colaboratory(又名Google Colab)是一个运行在Google Cloud上的在线笔记本环境。用户界面类似于Jupyter笔记本，它支持机器学习模型的GPU和TPU。在本教程中，我们将涵盖:

  • 如何创建和运行Colab笔记本单元格？
  • 如何从Google Drive读取数据？
  • 如何用Google Colab运行R编程语言？

我们开始吧！

第一步:创建一个Google Colab笔记本

步骤1.1 :登录你的Google账户。如果你没有谷歌帐户，去accounts.google.com创建一个帐户。

步骤1.2 :点击右上角的9点图标，选择驱动。

步骤1.3 :点击新建->-更多->-Google协同实验室打开一个新的Colab笔记本。

如果您在列表中没有看到谷歌协作实验室，请点击连接更多应用并在搜索栏中搜索谷歌协作实验室。

安装Google Colab，点击新建->-更多->-Google co laboratory打开一个新的Colab笔记本。

步骤1.4 :点击左上角的文件名，更改文件名。

步骤2(可选):设置运行时

第二步是设置运行时间。默认运行时间使用CPU，但是您可以通过点击运行时间->-更改运行时间类型来更改运行时间。

在笔记本设置弹出窗口中，我们可以将硬件加速器从无更改为 GPU 或 TPU 。GPU(图形处理器)和TPU(张量处理器)通常用于计算量大的模型。我们将把它保持为 None ，因为本教程不需要很高的计算能力。

步骤3:创建和运行单元

Google Colab笔记本中有两种类型的单元格，文本单元格和代码单元格。

步骤3.1 :添加新的文本单元格，将鼠标悬停在中间，直到 + 代码和+文本出现。点击+文本。

步骤3.2 :在新增的文本单元格中键入文本。您可以使用markdown来格式化文本，呈现的文本显示在右侧。当在单元格外单击时，单元格会自动呈现。

第3.3步:添加新的编码单元格，将鼠标悬停在中间，直到出现+编码和+文本为止。点击+代码。

步骤3.4 :在新添加的代码单元格中键入Python代码，点击运行按钮(一个黑色圆圈，里面有一个白色三角形)运行代码。这里我们输入2+3，得到5的结果。

步骤4:从Google Drive读取数据

当使用Google Colab时，从Google Drive读取数据是很常见的。在第四步中，我们将讨论如何连接Google Drive并读取数据。

步骤4.1 :通过运行下面的代码，将Google Drive挂载到Google Colab笔记本上。

# Mount Google Drive
from google.colab import drive
drive.mount('/content/drive')

将弹出一个窗口，询问“允许这台笔记本访问您的Google Drive文件吗？”选择蓝色的连接到Google Drive 。

然后选择用于Colab笔记本的Google账户，选择蓝色的允许。

我们将看到Google Drive成功连接后的输出Mounted at /content/drive。

步骤4.2 :用数据切换到Google Drive目录。

# Change directory
import os
os.chdir("drive/My Drive/contents/google_colab")

我们可以使用!pwd来检查当前目录。输出显示我们在正确的目录中。

步骤4.3 :将csv文件读入熊猫数据帧。现在Google Drive已经连接到了Colab笔记本，我们可以像从本地文件夹读取数据一样读取保存在Google Drive上的数据。

# Read in data
import pandas as pd
df = pd.read_csv('example_data.csv')
df.head()

步骤5:(可选):使用Google Colab进行R编程

要将Colab笔记本的默认语言从Python改为R，使用链接https://colab.to/r打开一个新页面。

要在同一个笔记本中同时使用Python和R，我们可以使用rpy2包。更多细节请参考我之前的教程如何用谷歌Colab笔记本使用R。
